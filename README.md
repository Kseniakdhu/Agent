Track: A + B

В этом репозитории реализован простой агент, который использует уже  sklearn-модель и при необходимости обращается к внешнему LLM-сервису (Яндекс/Алиса) через OpenAI‑совместимый клиент

Файлы:
- `iris_model.py` — обёртка над сохранённой моделью;
- `agent.py` — реализация агента;
- `Iris.ipynb` — ноутбук с обучением модели
- `Iris_model.pkl` - модель
- `REFLECTION.md` — эссе

Установка и подготовка:
1. (Рекомендуется) Создайте виртуальное окружение и установите зависимости:

python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```
